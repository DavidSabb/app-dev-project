# app-dev-project
## WealthWay
### Flutter based mobile application for finance/savings planning

# Key Features
## Dashboard Overview
Summaries of savings goals, progress, budget health, and upcoming targets

Simple and visual indicators for clarity

## Savings Goal

Track contributions over time.

Visual progress bars and goal completion metrics.

## Budget Planning

Manage income and expenses.

Categorize transactions.

Monthly summaries and budgeting insights.

## Analytics & Insights

Charts to show spending trends.

Daily, weekly, and monthly breakdowns.

Goal forecast based on contribution patterns.

## API Based Stocks

Purchase, Sell stocks fetched from APIs

Fetching Live Stock Prices
